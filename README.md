# O2O_exercise

Installation
------------
Download the project.
```bash
$ cd my_project/
$ composer install
$ bin/console server:start
```
Tests
-----
Execute this command to run tests:

```bash
$ cd my_project/
$ ./bin/phpunit
```

Documentation
-----
In route api/doc

Example:  http://127.0.0.1:8000/api/doc
